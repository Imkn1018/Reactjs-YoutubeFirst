## サイト概要

Youtube の API を用いて、Youtube クローン APP を作成しました。
元々動画コンテンツを視聴するのが趣味であったため、完成品を想像しつつ楽しみながら開発を進めることができました。

## 作成した目的

React の API 通信・グローバルステイト管理の実践編として作成しました。また、グリッドレイアウトやレスポンシブ対応などの CSS の基本的な使い方を身に付けることも狙いでした。

## 【開発環境】

- JavaScript
- React
- Styled Component
- Vercel
- Youtube API
